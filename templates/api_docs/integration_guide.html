Integration Guide

How to embed Kashima Research data into your website or app.

Basic Embed

Add search to your site with a simple form:

<form action="https://api.kashima.research/search" method="GET">
  <input name="q" placeholder="Search forks...">
  <button type="submit">Search</button>
</form>

JavaScript Widget

Load data dynamically:

fetch('/api/forks?search=Fox 36', {
  headers: { 'Authorization': 'YOUR_KEY' }
}).then(r => r.json()).then(data => {
  // display data.results
});

WordPress / CMS

Use our API to pull product data into your product pages or blog. Cache the results to avoid hitting rate limits.
