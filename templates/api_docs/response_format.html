Response Format

All responses come back as JSON. Here's how to read them.

Basic Response Structure

{
  "success": true,
  "count": 3,
  "data": [...]
}

Fork Object

{
  "id": "fox-36-2024",
  "model": "Fox 36 Float",
  "year": 2024,
  "travel": 150,
  "offset": 44,
  "axle": "QR15",
  "images": [...]
}

Response Fields

  • success - true or false
  • count - Number of results returned
  • data - The actual data you requested
  • error - Error message if success is false
