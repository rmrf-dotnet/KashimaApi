Authentication

Every request needs your API key. This identifies your account and tracks your usage.

Using Your API Key

Add your API key to the Authorization header:

Authorization: YOUR_API_KEY

Example: cURL

curl "https://api.kashima.research/forks?search=Fox+36" \
  -H "Authorization: kr_live_abc123xyz"

Example: JavaScript

const response = await fetch('https://api.kashima.research/forks?search=Fox+36', {
  headers: { 'Authorization': 'kr_live_abc123xyz' }
});

Example: Python

import requests

response = requests.get('https://api.kashima.research/forks?search=Fox+36',
  headers={'Authorization': 'kr_live_abc123xyz'})
