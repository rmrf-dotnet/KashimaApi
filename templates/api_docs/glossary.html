Glossary

Quick definitions for bike and suspension terms you'll see in the API.

Travel

How far the suspension can compress. More travel = more ability to absorb big hits. Measured in millimeters.

Offset

How far the axle line is from the crown. Affects handling. Common: 37, 40, 44, 51mm.

Axle

The thru-axle that holds the wheel. Common sizes: QR15 (15mm), Boost (110x15mm).

Eye-to-Eye

Shock length from one mounting eyelet to the other. Determines frame fit.

Stroke

How far the shock shaft moves. Combined with eye-to-eye to size the shock.

Compression Damping

Controls how fast the suspension compresses. More compression = firmer feel.

Rebound Damping

Controls how fast the suspension extends after a hit. Too fast = bouncy.
