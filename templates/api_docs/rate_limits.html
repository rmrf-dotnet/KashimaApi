Rate Limits

Rate limits prevent any one user from overwhelming the service. Here's what you need to know.

Free Tier Limits

  • 100 requests per day
  • 10 requests per minute

What Happens If You Exceed

You'll get a 429 error. Wait 60 seconds and your limit resets.

{"success": false, "error": "Rate limit exceeded", "retry_after": 60}

Best Practices

  • • Cache responses on your end
  • • Only request data you need
  • • Add delays between requests
  • • Consider upgrading for higher limits
