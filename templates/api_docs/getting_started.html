Getting Started

This guide walks you through signing up, getting your API key, and making your first request in just a few minutes.

Step 1: Sign Up

Create a free account at our registration page. No credit card required.

Create Free Account

Step 2: Get Your API Key

After signing up, your API key will be shown on your dashboard. Copy it somewhere safe - you'll need it for every request.

Your API key looks like: kr_live_abc123xyz...

Step 3: Make Your First Request

Now let's search for a Fox 36 fork. Replace YOUR_API_KEY with your actual key:

curl "https://api.kashima.research/forks?search=Fox+36" \
  -H "Authorization: YOUR_API_KEY"
