Error Handling

When something goes wrong, the API returns helpful error messages so you can fix the issue.

401 - Unauthorized

Your API key is missing or invalid.

{"success": false, "error": "Invalid API key"}

429 - Too Many Requests

You've exceeded your rate limit. Wait a moment and try again.

{"success": false, "error": "Rate limit exceeded"}

404 - Not Found

The endpoint or resource doesn't exist.

{"success": false, "error": "Endpoint not found"}

500 - Server Error

Something went wrong on our end. Try again in a few seconds.

{"success": false, "error": "Internal server error"}
