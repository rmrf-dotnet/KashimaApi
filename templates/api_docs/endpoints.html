API Endpoints

All endpoints are GET requests that return Fox suspension part data in JSON format.

/forks

Get all Fox suspension forks. Use query parameters to filter results.

GET /forks?search=Fox 36&filter=150mm

Returns: Array of fork objects with specs, measurements, and images.

/shocks

Get all Fox shock absorbers. Filter by type, size, or mount.

GET /shocks?search=X2&filter=230x60

Returns: Array of shock objects with stroke, eye-to-eye, and mount specs.

/droppers

Get all Fox dropper posts. Filter by travel, diameter, or lever style.

GET /droppers?search=Transfer&filter=150mm

Returns: Array of dropper post objects with travel and length specs.

Query Parameters

  • search - Search by model name
  • filter - Filter by parameter like travel or size
  • limit - Number of results (default 20)
  • offset - Pagination offset
