Getting Started

To use the API, include your API key in the request header or as a query parameter.

# Using header

curl -H "X-API-Key: YOUR_API_KEY" \

https://kashima-research-api/api/v1/parts

# Using query parameter

curl "https://kashima-research-api/api/v1/parts?api_key=YOUR_API_KEY"

Endpoints

GET /api/v1/parts

Get all parts. Optional query parameter: category=forks|shocks|droppers

GET /api/v1/parts/<category>/<model>

Get specific part details. Example: /api/v1/parts/forks/36

Response Format

{
  "forks": {
    "category": "forks",
    "models": {
      "36": {
        "name": "36",
        "series": "Factory",
        "stanchion_diameter": "36mm",
        "travel": ["140mm", "150mm", "160mm"],
        "wheel_size": ["27.5\"", "29\""],
        "damper": "GRIP X2",
        ...
      }
    }
  }
}
